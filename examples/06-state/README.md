# State

Demonstrates reactive state with `State[T]`, using `.Get()` in render and `.Set()`/`.Update()` to trigger automatic re-renders, plus `if`/`for`/`:=` bindings for reactive children.

![State](../../docs/public/guides/06.png)

## Run

```bash
go run .
```
