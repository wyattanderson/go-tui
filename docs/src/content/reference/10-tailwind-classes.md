Utility classes for the `class` attribute.

| Class | Description |
|-------|-------------|
| `flex` | Display flex (row direction) |
| `flex-col` | Display flex (column direction) |
| `grow` | Flex grow to fill space |
| `shrink-0` | Prevent flex shrink |
| `gap-N` | Gap of N characters between children |
| `p-N` | Padding of N on all sides |
| `px-N` | Horizontal padding of N |
| `py-N` | Vertical padding of N |
| `m-N` | Margin of N on all sides |
| `border-single` | Single line border ┌─┐│└─┘ |
| `border-double` | Double line border ╔═╗║╘═╝ |
| `border-rounded` | Rounded border ╭─╮│╰─╯ |
| `border-thick` | Thick border ┏━┓┃┗━┛ |
| `font-bold` | Bold text |
| `font-dim` | Dim/faint text |
| `font-italic` | Italic text |
| `font-underline` | Underlined text |
| `text-COLOR` | Text color (red, green, blue, cyan, magenta, yellow, white) |
| `bg-COLOR` | Background color |
| `items-center` | Align items center (cross axis) |
| `items-start` | Align items start |
| `items-end` | Align items end |
| `items-stretch` | Stretch items to fill |
| `justify-center` | Justify content center (main axis) |
| `justify-between` | Space between items |
| `justify-around` | Space around items |
| `justify-evenly` | Space evenly between items |
| `h-full` | Height 100% |
| `w-full` | Width 100% |
| `text-center` | Center text alignment |
| `text-right` | Right text alignment |
| `truncate` | Truncate overflowing text with ellipsis |
