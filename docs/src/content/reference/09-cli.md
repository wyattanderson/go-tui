Command-line tool.

## tui generate

```shell
tui generate [path...]
```

Generate Go code from .gsx files.

## tui check

```shell
tui check [path...]
```

Check .gsx files for errors without generating.

## tui fmt

```shell
tui fmt [--check] [--stdout] [path...]
```

Format .gsx files.

## tui lsp

```shell
tui lsp [--log FILE]
```

Start the language server (stdio transport).
