Flexbox layout constants and types.

## Direction

```go
Row | Column
```

Flex direction for child layout.

## Justify

```go
JustifyStart | JustifyCenter | JustifyEnd | JustifySpaceBetween | JustifySpaceAround | JustifySpaceEvenly
```

Main-axis content alignment.

## Align

```go
AlignStart | AlignCenter | AlignEnd | AlignStretch
```

Cross-axis item alignment.

## Value

```go
Fixed(n) | Percent(n) | Auto()
```

Dimension specification for width/height.

## BorderStyle

```go
BorderNone | BorderSingle | BorderDouble | BorderRounded | BorderThick
```

Border drawing styles with Unicode box characters.
