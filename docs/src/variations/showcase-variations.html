Three approaches to annotated code. All share: concept buttons, code highlighting, hover interaction.

variation a — description strip

Concept buttons sit in a bar (like the EditorSimulation feature tabs). When active, a description strip appears between the buttons and the code showing what the concept does. The code below highlights the matching lines. Clean vertical flow, no side column.
